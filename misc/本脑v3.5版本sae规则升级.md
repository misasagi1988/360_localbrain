升级说明：
万虎那边attack_port、victim_port属性类型改动，由string改为number数组，sae_rule、search_rule数据表部分字段需要更新。
其中，sae_rule需更新raw及statement字段，raw可通过sql更新，statement可通过调用recompile接口更新。
search_rule
更新sql语法:

```sql
CREATE TEMPORARY TABLE temp_sae_rule 
select id, rule_name, JSON_SET(raw, CONCAT('$.selects[', json_index-1, '].event.attrType'), 'number', CONCAT('$.selects[', json_index-1, '].event.array'), true) as raw, update_time 
FROM sae_rule,
     JSON_TABLE(JSON_EXTRACT(raw, '$.selects'), '$[*]' COLUMNS (
       json_index FOR ORDINALITY,
       attrName VARCHAR(255) PATH '$.event.attrName',
       attrType VARCHAR(255) PATH '$.event.attrType',
       attrField VARCHAR(255) PATH '$.event.attrField',
       array BOOLEAN PATH '$.event.array'
     )) AS jt
WHERE jt.attrField = 'attack_port';
UPDATE sae_rule AS sr INNER JOIN temp_sae_rule AS temp ON sr.id = temp.id SET sr.raw = temp.raw, sr.update_time = temp.update_time;
DROP TABLE temp_sae_rule
```
