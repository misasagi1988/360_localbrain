SIEM: 安全信息与事件管理平台 (Security Information Event Management )
SOC: 安全运营中心（Security Operation Center）
SIEM侧重于日志的集中式管理和审计，SOC则用于安全日志的分析和安全风险的监控与定位。两者的侧重点不同决定了，SIEM可以用产品来交附而SOC则必需加入MSS服务的人工干预来完善。

MDR: 托管检测与响应（Managed Detection and Response）
XDR: 扩展检测响应（Extended Detection and Response）
MSS: 托管安全服务（Managed Security Service）
DSP: 数据安全平台（Data Security Platforms）

UEBA: 用户行为分析
SOAR: 安全编排自动化响应
