# 一体机3.5版本功能

标签（空格分隔）： 一体机

---

## 版本发布时间
20220630发布

### 功能简介
- XDR化 
  - XDR 数据接入 
  - XDR 检测模型 
  - XDR 安全事件 
  - XDR 处置响应 
  - XDR 大屏：BDR大屏、EDR大屏、XDR全局大屏 
- UEBA集成（一体机适配版） 
- 实体画像 
- 其他优化 
  - 全量情报适配合入 

### 详细介绍
-  新功能：全面XDR功能设计与优化
1.	XDR数据源接入 
o	简化了360自有神经元的接入复杂度，内置的XDR数据源实现了一键配置能力，只需要输入少量信息即可完成对自有神经元的数据采集、SOAR实例创建
o	内置多种360自有神经元：AISA/NDR、天相、桌管、浏览器、防火墙、沙箱、蜜罐，支持一键接入。后续也将持续建设、添加其他自有神经元信息。
o	针对包含SOAR联动响应的神经元，支持一键创建应用实例
2.	XDR检测模型 
o	内置数500+ XDR开箱即用的异常检测模型。
o	内置模型覆盖所有支持的XDR数据源，后续也将通过内容包的方式不断更新、新增
3.	全新的安全事件 
o	针对老版本安全事件效果差、无法理解等问题，新版本重新设计XDR安全事件，再降低理解复杂度的同时简化了页面表达，让威胁分析、研判更高效
o	实体画像信息与安全事件深度融合，可以大幅度的提高威胁分析效率
o	优化供给链路图绘制，一定数据量下能够自动的优化显示样式，展示访问关系更为简洁易读；
4.	XDR大屏 
o	针对全面化的XDR设计，新增一张XDR专属大屏，大屏总数累计到8张

-  能力增强
1.	实体画像 
o	支持IP、域名、文件md5三类数据的行为画像，补齐与友商功能参数上的不足
o	支持拓线分析，无限拓展
2.	处置响应 
o	优化了SOAR的相关显示菜单，将重点菜单更加突出展示
o	针对告警联动SOAR预案不方便，新版本特别新增自动处置策略功能，通过配置全局的处置策略，可以在安全事件、告警发生时自动执行处置预案
3.	行为分析 
o	新增用户行为分析功能（UEBA Lite版），较为之前的异常行为分析，目前支持更多的分析模型和类型
o	内置10+行为分析模型，后续可通过升级包持续扩充
4.	告警白名单 
o	支持对误报一键加白过滤，大大简化了安全分析中的操作流程

- 功能优化 
1.	快捷菜单优化，支持一键进入实体画像；
2.	多网卡多IP环境，支持配置网关信息；
3.	多网卡多IP环境，支持选择IP作为业务访问IP;
4.	支持通过云端MDR平台访问注册至一体机的AISA、NDR、EDR
5.	增加hash威胁情报，本地黑情报类型扩增至九类
